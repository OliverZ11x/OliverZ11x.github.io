    基类语言模型接口¶

    语言模型的接口或基类，BaseLanguageModel，它定义了一系列与语言模型交互的方法。下面是对每个方法功能的解释：

    1. generate_prompt(prompts: List[PromptValue], stop: Optional[List[str]], callbacks: Callbacks): LLMResult
    2. 功能：根据提供的提示（prompts）生成一个提示，并可选地定义停止符（stop）和回调（callbacks），以产生语言模型的结果（LLMResult）。

    3. 参数：

      • prompts: 提示列表，用于引导语言模型的生成。
      • stop: 可选的停止符列表，指示生成何时停止。
      • callbacks: 生成过程中调用的回调函数集。

    4. predict(text: str): str

    5. 功能：给定一段文本（text），使用语言模型预测并返回接下来的文本。

    6. 参数：

      • text: 要进行预测的文本。

    7. predict_messages(messages: List[BaseMessage]): BaseMessage

    8. 功能：给定一系列消息（messages），使用语言模型预测并返回接下来的消息。

    9. 参数：

      • messages: 要进行预测的消息列表，每个消息都是BaseMessage的实例。

    10. get_num_tokens(text: str): int

    11. 功能：计算给定文本（text）的令牌（token）数量。

    12. 参数：

      • text: 要计算令牌数量的文本。

    13. get_num_tokens_from_messages(messages: List[BaseMessage]): int

    14. 功能：计算给定消息列表（messages）中的总令牌数量。

    15. 参数：

      • messages: 要计算令牌数量的消息列表，每个消息都是BaseMessage的实例。

    16. get_token_ids(text: str): List[int]

    17. 功能：将给定文本（text）转换为令牌ID的列表。
    18. 参数：
      • text: 要转换为令牌ID的文本。

    这些方法提供了与语言模型进行交互的基础，包括生成文本、预测接下来的文本或消息、计算令牌数量以及将文本转换为令牌ID等功能。
